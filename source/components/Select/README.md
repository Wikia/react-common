Basic usage:
```js
<Select
    options={[
        // use createOption(val, label) for these
        {value: 1, label: "label 1"},
        {value: 2, label: "label 2"},
        {value: 3, label: "label 3"},
    ]}
/>
```

Loading indicator:
```js
<Select
    placeholder="Loading..."
    loading
/>
```
